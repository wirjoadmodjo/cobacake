	- proses instalasi menggunakan cakephp 3.x
	- membuat plugin layout
		1. bin/cake bake plugin NamaPlugin
		2. load plugin /config/bootstrap.php
			Plugin::load('NamaPlugin', ['bootstrap' => false, 'routes' => true]);
		3. Tambahkan folder 
			NamaPlugin/src/Template
			NamaPlugin/src/Template/Layout
		4. Buat default layout
			NamaPlugin/src/Template/Layout/default.ctp
		5. load layout di controller
			public function initialize()
			{
				// ....
				$this->viewBuilder()->layout('NamaPlugin.default');
				//...
			}
